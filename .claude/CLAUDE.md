# KaijuBytes Portfolio Site

## Project
Static portfolio site for Orlando Martinez (Lanz) / KaijuBytes.
Deployed to GitHub Pages at kaijubytes.github.io.

## Tech Stack
Astro + Tailwind CSS + React (for interactive components) | GitHub Pages

## Rules
- Astro pages for static content, React components only when interactivity is needed
- Tailwind CSS only, no custom CSS frameworks
- CSS variables for theming (light/dark mode via `.dark` class)
- No emoji in content
- All external links: `target="_blank" rel="noopener"`
- Images go in `public/images/`
- Blog posts as markdown in `src/content/blog/`
